Telegram bot notifications

Get notified immediately when Anthropic refreshes your limits. Sends alerts for natural 5-hour window decays and scheduled weekly digests.

Surprise reset detection

Anthropic occasionally refreshes limits for everyone unexpectedly. The background tracker detects the drop and alerts you so you can resume building.

quick start

Set up Telegram notifications in three minutes.

The app encrypts your Admin API key and Telegram bot credentials using OS-level safeStorage (macOS Keychain / Windows DPAPI) before local persistence.
